# OS Module in python

## OS Module functions

Below are the list of functions in os module

  - Handling Current working directory

   ```bash
      os.getcwd() # No argument needs to be passed 
   ```

  - Handling changing current working directory

   ```bash
      os.chdir(path) # Pass the parameter inside the chdir() method
   ```

   - Handling Creation of directory

   ```bash
      os.mkdir(path)  # Required arguments to be passed

      os.mkdirs(path)  # Required arguments to be passed
   ```
